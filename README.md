# Scripts

Collection of utility scripts for development workflows.

## Subfolders

### [minimax-statusline](./minimax-statusline/)
Custom statusline for Claude Code when using the MiniMax API. Includes quota tracking, China peak/off-peak hours indicator, and session cost.

### [claude-statusline](./claude-statusline/)
Custom statusline for Claude Code when using the standard Anthropic API. Lightweight version without MiniMax-specific dependencies.
